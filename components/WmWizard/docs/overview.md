# Overview

The Wizard component is a container element that guides users through a multi-step process or workflow within a mobile application. It provides a structured interface for breaking down complex tasks into manageable steps, making it ideal for onboarding sequences, form submissions, setup processes, or any scenario where users need to complete sequential actions while maintaining context and progress awareness.

This component offers a comprehensive step-by-step navigation experience with built-in progress tracking and flexible user interaction controls. It's particularly useful for mobile applications where screen real estate is limited and complex processes need to be simplified into digestible chunks.

## Features

- **Flexible Step Navigation** - Navigate between steps using customizable next, previous, and done buttons
- **Progress Tracking** - Display user progress with configurable progress indicators and titles
- **Customizable Actions** - Configurable button labels and alignment options for navigation controls
- **Cancellation Support** - Optional cancel functionality with customizable cancel button labels
- **Default Step Configuration** - Set initial starting step for the wizard sequence
- **Progress Visualization** - Multiple progress display types to match your app's design requirements
- **Content Flexibility** - Support for any child components within each wizard step
- **Mobile-Optimized** - Designed specifically for mobile user interface patterns and interactions
