# Overview

The Currency widget is a specialized input component designed for handling monetary values with proper currency formatting and validation. It automatically formats numbers according to the specified currency type and provides built-in validation for monetary data entry.

This component extends the base number input functionality with currency-specific features like symbol display, decimal place control, and localized formatting. It's ideal for applications requiring financial data input such as e-commerce, banking, or accounting applications.

## Features

- **Currency Formatting**: Automatic formatting based on selected currency type
- **Decimal Control**: Configurable decimal places with default 2-digit precision
- **Value Validation**: Built-in min/max value constraints and required field validation
- **Input Controls**: Support for step increments and custom validation patterns
- **Accessibility**: Full accessibility support with ARIA labels and roles
- **Real-time Updates**: Two-way data binding with configurable update triggers
- **Responsive Design**: Adaptive layout with device-specific visibility controls
