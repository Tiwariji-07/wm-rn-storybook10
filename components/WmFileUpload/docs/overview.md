# Overview

The FileUpload component provides a user-friendly interface for selecting and uploading files in mobile applications. It renders as a button with customizable icon and caption that opens the device's file picker when tapped.

This component is designed specifically for mobile file selection workflows, allowing users to choose files from their device's storage, camera, or other available sources depending on the platform capabilities.

## Features

- **File Selection**: Opens native file picker for selecting files from device storage
- **Customizable Appearance**: Configurable icon, caption, and styling options
- **Accessibility Support**: Built-in accessibility features with customizable labels and roles
- **Content Type Filtering**: Ability to restrict file types through content type specification
- **Touch Optimization**: Native touch interactions with optional touch effect customization
- **Responsive Design**: Adaptive display based on device screen sizes
- **Validation Support**: Pre-selection validation through callback events
