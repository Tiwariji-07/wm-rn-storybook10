# Overview

The `radioset` component provides a grouped collection of radio buttons that allows users to select a single option from multiple choices. Built on React Native, it offers flexible data binding capabilities and responsive layout options for mobile applications.

The component automatically generates radio buttons based on a provided dataset and supports various customization options including layout configuration, accessibility features, and custom rendering through partial templates.

## Features

- **Data Binding**: Bind to datasets to automatically generate radio button options
- **Responsive Layout**: Configure items per row for different screen sizes (xs, sm, md, lg)
- **Accessibility**: Built-in accessibility support with customizable labels and roles
- **Custom Rendering**: Use partial templates for custom item rendering
- **Flexible Data Sources**: Support for arrays, objects, and live variables
- **Validation Support**: Integration with form validation systems
- **Touch Optimized**: Native mobile touch interactions and feedback
