# Styling

## Available CSS Classes

The webview component provides the following CSS class for customization:

### Component-Specific Classes

- **`.app-webview`** - Default style class applied to the webview component. This class provides the base styling and layout properties for the embedded web content container.

## Standard App Classes

In addition to the component-specific class, you can apply standard application CSS classes to further customize the appearance and behavior of the webview component.
