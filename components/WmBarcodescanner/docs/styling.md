# Styling

## CSS Classes

The barcodescanner component provides the following CSS classes for customization:

### Component-Specific Classes

| Class Name | Description |
|------------|-------------|
| `app-barcodescanner` | Default style class applied to the barcode scanner component |

### Class Descriptions

**app-barcodescanner**
- Applied to the root element of the barcode scanner component
- Controls the overall appearance and layout of the scanner interface
- Can be used to customize dimensions, positioning, borders, and visual styling of the scanner viewport

### Additional Styling

Beyond the component-specific classes, you can apply standard application CSS classes and custom styles to further customize the barcode scanner's appearance and behavior.
