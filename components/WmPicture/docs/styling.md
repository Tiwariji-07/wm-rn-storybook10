# Styling

## Available CSS Classes

The picture component provides the following CSS classes for styling:

### Component-Specific Classes

- **`.app-picture`** - Default style class applied to the picture component
- **`.rounded-image`** - Applies rounded corners to the image for a softer, more modern appearance
- **`.thumbnail-image`** - Styles the image as a thumbnail with appropriate sizing and proportions for preview or gallery contexts

### Additional Styling

In addition to the component-specific classes listed above, you can apply standard application CSS classes to further customize the appearance and behavior of the picture component.
