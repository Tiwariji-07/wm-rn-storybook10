# Overview

The Picture component is a versatile image display widget designed for React Native applications. It provides comprehensive image handling capabilities including loading, displaying, and managing various image sources with built-in accessibility support and responsive design features.

The Picture component can be used standalone or integrated within other components like Live Lists, Data Grids, or Carousels. It supports various image sources including local files, URLs, and can be bound to database columns containing image resource locations.

## Features

- **Multiple Image Sources**: Support for local files, URLs, and database-bound image sources
- **Shape Customization**: Built-in support for rounded, circular, and thumbnail shapes
- **Responsive Design**: Configurable aspect ratios and resize modes for different screen sizes
- **Loading States**: Skeleton loading with customizable dimensions and fast load options
- **Accessibility**: Full accessibility support with customizable labels and roles
- **Touch Interactions**: Comprehensive touch event handling including tap, double-tap, and long-tap
- **Performance Optimization**: Lazy loading and device-specific visibility controls
- **SVG Support**: Native SVG image handling capabilities
