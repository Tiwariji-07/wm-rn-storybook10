# Styling

## Available CSS Classes

### Component-Specific Classes

| Class Name | Description |
|------------|-------------|
| `app-pie-chart` | Default style class for the pie chart component |

The `app-pie-chart` class provides the base styling for pie chart components, including default dimensions, spacing, and visual presentation of the chart container.

## Additional Styling Options

In addition to the component-specific class, you can use standard application CSS classes for common styling needs such as margins, padding, colors, and responsive behavior.
