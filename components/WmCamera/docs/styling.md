# Styling

## Available CSS Classes

The camera component provides the following specific styling class:

### `.app-camera`
- **Description**: Default style class for the camera component
- **Visual Effect**: Applies the base styling for camera functionality, including container layout, control positioning, and default appearance for camera interface elements

## Additional Styling Options

Beyond the component-specific class, you can also apply standard application CSS classes to customize the camera component's appearance and behavior according to your design requirements.
