# Overview

The Icon component is a fundamental UI element designed to display scalable vector icons within mobile applications. It serves as a versatile building block for creating intuitive user interfaces by providing visual representations of actions, objects, or concepts that enhance user experience and navigation.

This component extends beyond basic icon display by offering comprehensive customization options including animations, positioning, sizing, and interactive capabilities. Whether used standalone or combined with text captions, the Icon component provides the flexibility needed to create engaging and accessible mobile interfaces while maintaining consistent visual styling across your application.

## Features

- **Flexible Icon Sizing** - Customizable icon dimensions to fit various UI contexts and design requirements
- **Animation Support** - Built-in animation capabilities with configurable timing and iteration controls
- **Interactive Touch Handling** - OnTap event support for creating clickable icon buttons and interactive elements
- **Positioning Control** - Adjustable icon positioning relative to associated content or captions
- **Text Caption Integration** - Optional caption display for enhanced accessibility and user guidance
- **Custom Styling** - Icon class customization for theme-based styling and visual consistency
- **Skeleton Loading States** - Configurable skeleton placeholders with custom dimensions for improved perceived performance
- **Animation Timing Control** - Precise control over animation delays and iteration counts for sophisticated visual effects
