# Overview

The Icon component is a versatile UI element that displays graphical icons within mobile applications. It supports both font-based icons (wavicon and font-awesome) and image-based icons, making it suitable for enhancing user interface elements with visual cues and interactive elements.

Icons can be used standalone or integrated into other components like buttons and panels. The component provides extensive customization options including size control, positioning, animations, and touch interactions.

## Features

- **Multiple Icon Types**: Support for wavicon (mobile-optimized thin fonts), font-awesome icons, and custom image icons
- **Flexible Sizing**: Configurable icon dimensions with support for different size units
- **Touch Interactions**: Built-in support for tap, double-tap, long-tap, and touch events
- **Animations**: CSS-based animations including spin and pulse effects
- **Accessibility**: Full accessibility support with customizable labels and roles
- **Responsive Design**: Device-specific visibility controls and skeleton loading states
- **RTL Support**: Right-to-left layout compatibility
- **Caption Support**: Optional text labels alongside icons
