# Overview

The Card component is a versatile data display container designed to present structured information in a visually appealing and organized manner within mobile applications. It serves as a fundamental building block for creating content-rich interfaces, allowing developers to showcase various types of data, media, and interactive elements in a cohesive card-based layout that enhances user experience and content discoverability.

This component extends the BaseProps class and provides a flexible foundation for displaying complex data structures with embedded actions, navigation links, and hierarchical content organization. Cards are particularly effective for presenting collections of related information, such as product listings, user profiles, news articles, or dashboard widgets, making them an essential component for modern mobile app interfaces.

## Features

- **Flexible Content Container** - Supports custom child components and nested content structures
- **Interactive Actions** - Built-in support for action buttons and clickable elements
- **Navigation Integration** - Seamless linking capabilities with customizable item links
- **Icon Support** - Comprehensive icon system with customizable styling and positioning
- **Badge System** - Visual indicators and status badges for enhanced information hierarchy
- **Active State Management** - Dynamic styling for selected or active card states
- **Nested Content Support** - Ability to contain child items and create hierarchical layouts
- **Media Integration** - Configurable image display with adjustable height settings
- **Custom Styling** - Flexible icon class customization for brand consistency
- **Responsive Design** - Optimized layout that adapts to various screen sizes and orientations
