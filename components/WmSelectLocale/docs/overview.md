# Overview

The Select Locale component allows users to select their preferred language/locale in a mobile application, enabling localization and internationalization features. It extends the base select functionality specifically for locale selection and switching.

The component automatically integrates with WaveMaker's localization system, providing a seamless way for users to change the application language. It can detect browser preferences, allow manual selection, or be programmatically controlled.

## Features
- Automatic locale detection from browser preferences
- Integration with WaveMaker's localization system
- Support for custom locale datasets
- Configurable display expressions for locale names
- Accessibility support with ARIA attributes
- Touch-friendly interface for mobile devices
- Real-time locale switching capabilities
