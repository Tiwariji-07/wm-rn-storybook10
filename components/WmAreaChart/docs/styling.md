# Styling

## Available CSS Classes

The area-chart component provides the following specific CSS classes for customization:

### `.app-area-chart`
- **Description**: Default style class
- **Visual Effect**: Applies the base styling for the area chart component, including default colors, spacing, and layout properties

## Additional Styling Options

If additional customization is needed beyond the provided classes, you can use standard application CSS classes or create custom styles targeting the area chart component structure.
