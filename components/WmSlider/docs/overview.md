# Overview

The Slider component is an interactive input control that allows users to select a single value or range of values by dragging a handle along a horizontal track. This component is ideal for mobile applications where users need to adjust settings, filter content by numeric ranges, or input values within defined boundaries through an intuitive touch-based interface.

Built on the BaseDatasetProps foundation, the Slider component provides a highly configurable and accessible way to capture numeric input while offering immediate visual feedback to users about their selections and available value ranges.

## Features

- **Range Selection**: Supports both single value and range selection modes for flexible input scenarios
- **Customizable Bounds**: Configurable minimum and maximum values to define the selectable range
- **Stepped Values**: Adjustable step increments for precise value control and alignment with business requirements
- **Data Type Support**: Compatible with various numeric data types for seamless integration with different data models
- **Visual Markers**: Optional markers along the slider track to highlight important values or intervals
- **Interactive Tooltips**: Contextual tooltips that display current values with customizable positioning
- **Dynamic Expressions**: Support for custom tooltip expressions to display formatted or calculated values
- **Marker Labels**: Customizable text labels for slider markers to provide additional context
- **Mobile Optimized**: Touch-friendly interface designed for optimal mobile user experience
