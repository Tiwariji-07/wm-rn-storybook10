# Overview

The **Text widget** is a foundational input component that accepts single-line text input from users. It provides a comprehensive set of features for text input validation, formatting, and user interaction in mobile applications.

As an editor widget, the Text component offers extensive customization options for different input types, validation rules, and display formats, making it suitable for various use cases from simple text entry to complex formatted input fields.

## Features
- **Multiple Input Types**: Supports text, email, password, number, and other specialized input types
- **Client-side Validation**: Built-in support for regular expressions and character limits
- **Text Formatting**: Display format masks and auto-capitalization options
- **Accessibility Support**: Comprehensive accessibility features including labels and roles
- **Two-way Data Binding**: Real-time data synchronization with form models
- **Auto-completion**: Built-in autocomplete functionality
- **Floating Labels**: Modern material design-style floating label support
- **Skeleton Loading**: Loading state management with customizable skeleton dimensions
