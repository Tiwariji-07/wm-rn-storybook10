# Overview

The Text component is a versatile input field designed for mobile applications, providing users with a clean and intuitive way to enter text-based information. Built on the BaseInputProps foundation, this component serves as the primary interface for collecting user input such as names, messages, search queries, and form data throughout your application.

This component combines essential text input functionality with modern UX patterns, offering developers fine-grained control over input behavior and presentation. It seamlessly integrates with mobile keyboards and provides enhanced user experience through intelligent input handling and visual feedback mechanisms.

## Features

- **Floating Label Animation** - Dynamic label that moves above the input field when focused or filled
- **Placeholder Text Support** - Contextual hints to guide user input expectations
- **Skeleton Loading States** - Customizable loading placeholders with configurable dimensions
- **Smart Capitalization** - Automatic text capitalization based on input context
- **Keyboard Integration** - Custom return key types and submission handling
- **Auto-focus Capability** - Programmatic focus control for improved user flow
- **Autocomplete Support** - Integration with system-level autocomplete suggestions
- **Text Trimming** - Automatic whitespace removal for cleaner data input
- **Mobile-First Design** - Optimized for touch interfaces and mobile interaction patterns
