# Styling

The confirmdialog component provides the following CSS classes for customization:

## CSS Classes

| Class Name | Description |
|------------|-------------|
| `app-confirmdialog` | Default style class applied to the confirm dialog component |
| `app-confirmdialog-rtl` | Style class for right-to-left (RTL) text direction support, adjusting layout and positioning for RTL languages |

## Usage

These classes can be used to customize the appearance and layout of the confirm dialog component. The default class provides the base styling, while the RTL class ensures proper visual presentation when the component is used in right-to-left language contexts.
