# Overview

The Checkbox component is a two-state input control that allows users to select or deselect an option. It's commonly used for binary choices, agreements, or multi-selection scenarios in mobile applications. The component provides a simple yet powerful interface for capturing user preferences and boolean data.

Checkboxes are ideal for scenarios such as accepting terms and conditions, enabling settings, selecting multiple items from a list, or any situation where users need to make binary choices. The component supports customizable checked and unchecked values, making it flexible for different data requirements.

## Features
- Binary state management (checked/unchecked)
- Customizable checked and unchecked values
- Caption display support
- Read-only and disabled states
- Touch-friendly interaction
- Accessibility support with hints
- Responsive visibility controls
- Skeleton loading state
- Custom styling support
- Event-driven interactions
