# Callback Events

| Event | Description | Parameters |
|-------|-------------|------------|
| `onChange` | Triggered when the carousel slide changes | `(this, index, prevIndex)` - Current component instance, new slide index, previous slide index |
| `onTap` | Triggered when a carousel slide is tapped | `(e)` - Touch event object |
| `onTouchstart` | Triggered when touch interaction starts | `(e)` - Touch start event object |
| `onDoubletap` | Triggered when a slide is double-tapped | `(e)` - Double tap event object |
| `onLongtap` | Triggered when a slide is long-pressed | `(e)` - Long tap event object |
| `onTouchend` | Triggered when touch interaction ends | `(e)` - Touch end event object |
