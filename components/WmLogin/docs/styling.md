# Styling

The login component provides the following CSS classes for customization:

## Available CSS Classes

### `.app-login`
Default style class for the login component container. This class controls the overall appearance and layout of the login form.

### `.app-login-username`
Styles the username input field within the login component. This class can be used to customize the appearance of the username field, including styling for borders, padding, colors, and focus states.

### `.app-login-password`
Styles the password input field within the login component. This class can be used to customize the appearance of the password field, including styling for borders, padding, colors, and focus states.

## Usage

You can override these classes in your CSS to customize the login component's appearance:

```css
.app-login {
  /* Custom container styles */
}

.app-login-username {
  /* Custom username field styles */
}

.app-login-password {
  /* Custom password field styles */
}
```
