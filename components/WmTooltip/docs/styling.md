# Styling

## CSS Classes

The tooltip component provides the following CSS class for styling customization:

### Component-Specific Classes

- **`.app-tooltip`** - Default style class that applies the base styling for the tooltip component, including positioning, background, typography, and visual appearance such as borders, shadows, and z-index layering.

## Additional Styling

You can also apply standard application CSS classes and custom styles to further customize the tooltip's appearance, positioning, and behavior to match your design requirements.
