# Styling

## CSS Classes

The alertdialog component provides the following CSS class for styling:

### `.app-alertdialog`
- **Description**: Default style class
- **Visual Purpose**: Provides the base styling for the alert dialog component, including default appearance, positioning, and layout properties

## Additional Styling Options

Beyond the component-specific class, you can utilize standard application CSS classes and custom styles to further customize the alertdialog's appearance, spacing, colors, and interactive states.
