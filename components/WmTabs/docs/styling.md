# Styling

## Available CSS Classes

The tabs component provides the following specific styling classes:

### `.app-tabs`
Default style class that applies the standard appearance and layout for the tabs component.

### `.tabs-with-arrow-indicator`
Adds arrow-style visual indicators to the tabs, typically displaying directional arrows to enhance navigation visual cues.

## Additional Styling

Beyond these component-specific classes, you can apply standard application CSS classes for further customization of spacing, colors, typography, and other visual properties.
