# Callback Events

| Event | Description |
|-------|-------------|
| `onClose` | Triggered when the date picker modal is closed, regardless of whether a date was selected or the action was cancelled |
| `onSelect` | Triggered when the user confirms a date selection. Receives the selected Date object as a parameter |
| `onCancel` | Triggered when the user presses the cancel button or dismisses the picker without selecting a date |
