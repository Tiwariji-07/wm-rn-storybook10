# Overview

The Spinner component is a fundamental UI element designed to provide visual feedback to users during loading states, data processing, or asynchronous operations in mobile applications. This versatile component helps maintain user engagement by clearly communicating that the application is actively working, preventing user confusion during wait times and improving overall user experience.

Built on the BaseProps foundation, the Spinner component offers multiple visual representation options to seamlessly integrate with your application's design language. Whether you need a simple animated icon, custom imagery, or sophisticated Lottie animations, this component provides the flexibility to create engaging loading experiences that align with your brand identity while maintaining consistent behavior across different screen sizes and orientations.

## Features

- **Multiple Animation Types**: Support for icon-based spinners, custom images, and Lottie animations
- **Flexible Sizing**: Configurable icon sizes and custom image dimensions (width/height)
- **Icon Customization**: Custom icon classes for personalized spinner appearances
- **Caption Support**: Optional text labels to provide context during loading states
- **State Management**: Disabled state functionality for controlling spinner behavior
- **Unique Identification**: ID and name properties for component tracking and accessibility
- **Responsive Design**: Adaptive sizing that works across different device screen sizes
- **Performance Optimized**: Lightweight rendering suitable for mobile environments
