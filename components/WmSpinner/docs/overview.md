# Overview

The **Spinner** component provides visual feedback to users during loading states and background operations. It displays an animated icon or image with an optional caption to indicate that the application is processing data or performing tasks.

This component is essential for maintaining good user experience by preventing users from thinking the application has frozen or stopped working during asynchronous operations.

## Features

- Multiple display types (icon, image, or Lottie animation)
- Customizable loading messages
- Variable tracking for automatic show/hide behavior
- Flexible sizing and styling options
- Device-specific visibility controls
- RTL language support
- Touch effect customization
