# Styling

## Available CSS Classes

The donut chart component provides the following CSS class for customization:

### Component-Specific Classes

- **`.app-donut-chart`** - Default style class that applies the base styling for the donut chart component, including layout, positioning, and default visual appearance of the chart container.

## Additional Styling Options

Beyond the component-specific class, you can utilize standard application CSS classes and custom styling to further customize the appearance of your donut chart, including colors, dimensions, animations, and responsive behavior.
