# Styling

## CSS Classes

The bar-chart component provides the following specific CSS classes for styling:

### `.app-bar-chart`
**Default style class** - The primary CSS class applied to the bar chart component. This class defines the base styling and layout properties for the chart container, including dimensions, spacing, and default visual presentation.

## Additional Styling

You can also apply standard application CSS classes to further customize the appearance of your bar chart component according to your design system requirements.
