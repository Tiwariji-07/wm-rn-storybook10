# Callback Events

| Event | Description |
|-------|-------------|
| `onFocus` | Triggered when the select widget gains focus. Parameters: `(event, widget)` |
| `onBlur` | Triggered when the select widget loses focus. Parameters: `({}, widget)` |
| `onChange` | Triggered when the selected value changes. Parameters: `(undefined, widget, newValue, oldValue)` |
| `onTap` | Triggered when the widget is tapped. Parameters: `(event)` |
| `onTouchstart` | Triggered when a touch gesture begins on the widget. Parameters: `(event)` |
| `onDoubletap` | Triggered when the widget is double-tapped. Parameters: `(event)` |
| `onLongtap` | Triggered when a long tap gesture is performed. Parameters: `(event)` |
| `onTouchend` | Triggered when a touch gesture ends. Parameters: `(event)` |
