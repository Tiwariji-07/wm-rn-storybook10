# Styling

## Available CSS Classes

### Component-Specific Classes

| Class Name | Description |
|------------|-------------|
| `app-line-chart` | Default style class for the line chart component |

The `app-line-chart` class provides the base styling for the line chart component, including default dimensions, colors, and layout properties for the chart container and visual elements.

## Additional Styling Options

In addition to the component-specific class, you can apply standard application CSS classes for common styling needs such as spacing, colors, and responsive behavior.
